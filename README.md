# Risk-to-Resilience

## Project Overview
Risk-to-Resilience is an AI and data analytics based platform designed to analyze country safety and predict future stability trends using global datasets and machine learning techniques.

## Features
- Country safety analysis
- Predictive trend forecasting
- Data visualization dashboard
- AI-powered insights
- Interactive frontend interface

## Tech Stack

### Frontend
- React.js
- HTML/CSS
- JavaScript

### Backend
- Python
- Flask
- Machine Learning Libraries
